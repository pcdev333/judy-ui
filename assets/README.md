# Assets

Place the following image files in this directory:

| File | Size | Usage |
|---|---|---|
| `icon.png` | 1024×1024 | App icon (iOS + Android) |
| `splash.png` | 1242×2688 | Splash screen |
| `adaptive-icon.png` | 1024×1024 | Android adaptive icon |
| `favicon.png` | 48×48 | Web favicon |

Brand color: `#E8470A` (Judy orange)

## Generating placeholder assets (development only)
```bash
node scripts/generate-assets.js
```
